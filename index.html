Introduction

JavaScript is a versatile programming language commonly used to create interactive effects within web browsers. Created by Netscape in 1995, it was initially intended to add small-scale interactivity to web pages. Over time, JavaScript has evolved into a powerful, versatile, and efficient language capable of handling complex logic, animations, and event-driven functionalities. With the advent of JavaScript engines like V8 and Node.js, it's now used on both the front end (in the browser) and the back end (on servers), making it essential for web development and a core technology alongside HTML and CSS.

What you should already know

Before diving into JavaScript, some prerequisites will help you understand concepts more easily:

  • HTML (HyperText Markup Language): It's beneficial to know the basics of HTML, as JavaScript often interacts with HTML elements to create dynamic web pages.
  • CSS (Cascading Style Sheets): A fundamental understanding of CSS will help, as JavaScript can manipulate styles to change the appearance of elements on a page.
  • Basic Programming Concepts: Familiarity with basic concepts such as variables, loops, functions, and conditional statements will be advantageous. Although JavaScript introduces unique concepts, many principles are universal.
JavaScript and Java

JavaScript and Java are two distinct programming languages with different syntaxes, use cases, and behaviors, despite having similar names. JavaScript is dynamically typed, prototype-based, and primarily used for client-side scripting. Java, on the other hand, is a statically typed, object-oriented programming language used in server-side applications, Android development, and large-scale systems. While Java must be compiled before running, JavaScript code is interpreted, making it faster for smaller applications. Knowing the differences can help avoid confusion between the two languages and clarify that learning one does not directly translate to learning the other.

Hello world

The classic "Hello, World!" example is often used as a simple introduction to a programming language. In JavaScript, you can write this in various ways to display a message on a web page or in the browser console. For example: 

          
          // Your code here
          function example() {
          console.log("This is a block of code.");
          }

In this example, JavaScript waits until the HTML document has fully loaded, then displays "Hello, World!" inside the div element.

Variables

In JavaScript, variables are used to store data values. These values can be modified and accessed throughout the code, making variables fundamental to any programming. JavaScript has three main ways to declare variables:

  • var: This is the oldest way to declare a variable in JavaScript, with function scope rather than block scope.
  • let: Introduced in ES6 (ECMAScript 2015), let allows you to declare block-scoped variables, making it safer for use in loops and conditionals.
  • const: Also introduced in ES6, const declares a constant, read-only variable, which means its value cannot be changed after initial assignment.
Declaring variables

Declaring variables in JavaScript requires a keyword (var, let, or const) followed by the variable name. Here are examples: 

          
          var name = "Alice";     // Declares a function-scoped variable
          let age = 25;           // Declares a block-scoped variable
          const country = "USA";  // Declares a block-scoped constant

Using let and const is now the preferred approach for better control of scope. const should be used for values that remain constant, like configuration values or mathematical constants.

Variable scope

Scope defines where variables can be accessed or used in JavaScript. JavaScript has three types of scope:

  • Global Scope: Variables declared outside any function or block have a global scope, meaning they can be accessed anywhere in the code.
  • Function Scope: Variables declared inside a function are accessible only within that function.
  • Block Scope: Variables declared with let or const within a block (such as in loops or conditionals) are only accessible within that block. This was introduced to prevent issues like unexpected variable modification in loops.
Global variable

Global variables are accessible throughout the entire codebase, making them useful for values that need to be referenced frequently. However, overusing global variables can lead to code conflicts and bugs, especially in larger applications. In modern JavaScript, global variables should be minimized and, if needed, encapsulated to prevent unexpected behavior.

Constants

Constants are variables that cannot be reassigned after they are set. They provide a way to declare values that should remain unchanged, which can help prevent errors in your code. JavaScript uses const to declare constants, which are block-scoped: 

          
            const MAX_USERS = 100;
            const API_URL = "https://api.example.com";

Once declared, attempting to reassign a constant will result in an error.

Data types

JavaScript supports multiple data types, which can be loosely categorized into two types: Primitive Types and Reference Types.

Primitive Types include:

  • String: "Hello"
  • Number: 42 or 3.14
  • Boolean: true or false
  • Undefined: A declared variable without an assigned value
  • Null: A deliberate non-value
  • Symbol: A unique identifier (introduced in ES6)

Reference Types include:

  • Objects: A collection of key-value pairs
  • Arrays: An ordered list of values
If else statement

The if-else statement is a control structure that allows you to execute code based on a condition: 

          
            let temperature = 30;
            if (temperature > 25) {
                console.log("It's hot outside.");
            } else if (temperature < 15) {
                console.log("It's cold outside.");
            } else {
                console.log("The weather is mild.");
            }

The if block runs if the condition is true; otherwise, the else block runs. Additional conditions can be checked with else if statements.

While statement

The while loop repeatedly executes a block of code as long as a specified condition evaluates to true: 

          
            let count = 0;
            while (count < 5) {
                console.log("Count is: " + count);
                count++;
            }

In this example, the while loop increments count and prints it, stopping once count reaches 5.

Function declarations

Functions are reusable blocks of code that can be called to perform a specific task. Declaring functions in JavaScript is done using the function keyword: 

          
            function multiply(a, b) {
                return a * b;
            }
            console.log(multiply(2, 3)); // Output: 6

JavaScript functions allow parameter passing and return values, making them versatile for code organization. Functions can also be assigned to variables, passed as arguments, and returned from other functions.
